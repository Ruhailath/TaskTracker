Task Tracker

Version: 1.0.0

Task Tracker is a simple and user-friendly web application built using Angular. It allows users to manage their daily tasks by adding, updating, completing, and deleting tasks efficiently.

The application uses a mock backend powered by JSON Server, where db.json acts as the database. RESTful APIs are used to perform all CRUD operations.

This project was developed as part of an internship assignment to demonstrate frontend development skills, API integration, and routing using Angular.
